#### 常用的块级标签

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>块级标签</title>
</head>
<body>
<!-- 标题标签 h1~h6 -->
    <h1>标题标签1</h1>
    <h2>标题标签2</h2>
    <h3>标题标签3</h3>

<!-- 段落标签 p -->
    <p>段落标签</p>

<!-- 有序列表标签 ol -->
    <ol>
        <li>有序1</li>
        <li>有序2</li>
    </ol>

<!-- 无序列表标签 ul -->
    <ul>
        <li>无序1</li>
        <li>无序2</li>
    </ul>

<!-- 定义描述标签 dl -->
    <dl>
        <dt>咖啡</dt>
        <dd>一种黑色的热饮料，原料据说是咖啡豆，非洲盛产这种原料</dd>
        <dd>可以提神，刺激神经</dd>
    </dl>

<!-- 表格标签 table -->
    <table>
        <tr>
            <td>百度</td>
            <td>新浪</td>
        </tr>

    </table>

<!-- 表单 form -->
    <form>

    </form>

<!-- 分区标签 div -->
    <div>分区标签</div>
</body>
</html>
```

#### a标签

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>a标签</title>
</head>
<body>
<a href="块级标签.html" target="_blank">块级标签</a>
</body>
</html>
```

#### 特殊符号

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>特殊符号</title>
</head>
<body>
    大于：&gt;<br/>
    小于：&lt;<br/>
    版权号：&copy;<br/>
    空格：a&nbsp;b<br/>
    引号：&quot;
</body>
</html>
```
